# Video Relevancia

Este proyecto entrena un modelo para detectar frames relevantes en un video usando histogramas y un clasificador RandomForest.

## 📂 Estructura
- `requirements.txt` → dependencias necesarias.
- `config.py` → parámetros globales (videos, carpetas, flags).
- `utils.py` → funciones auxiliares para procesar videos y extraer histogramas.
- `main.py` → flujo principal: entrenamiento y predicción.
- `README.md` → documentación del proyecto.

   ```bash
   pip install -r requirements.txt
